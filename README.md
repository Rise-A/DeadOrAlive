# Dead Or Alive #
This is the GitHub repository for the game, "Dead Or Alive", a fast paced "Guess Who?" style game, where players have to guess who among the crowd is the WANTED person. The game utilizes kNN search, which calculates a similarity score according to the player's description of the WANTED person, highlighting the person with the highest similarity score.

## Technologies Used ##
* C#
* Unity 2D
